# Uptick Origin Testnet – v0.3.2 Upgrade Announcement

Dear Uptick Origin Validators,

We are excited to announce that the Uptick Origin testnet will be upgraded to v0.3.2：
- Target Block Height: 12819990
- Estimated Time: April 29, 2026, at 15:00 UTC
- Commit Hash: bde718b6b1def40da9bd79cf9872b8f1ea0b2e7d

##  For Validators Running Cosmovisor

Please refer to our [Cosmovisor](https://upticknft.gitbook.io/uptick-network-documentation/guides/quickstart/cosmovisor) Setup Guide.

Please Note: Cosmovisor will perform a full backup unless you set the environment variable: UNSAFE_SKIP_BACKUP=true

```bash
# make a dir if you've not
mkdir -p $DAEMON_HOME/cosmovisor/upgrades/v0.3.2/bin

#Download the source code and build it
git clone https://github.com/UptickNetwork/uptick.git
cd uptick
git checkout v0.3.2
make build

#Copy binary files
cp ./build/uptickd $DAEMON_HOME/cosmovisor/upgrades/v0.3.2/bin/

# View version should be v0.3.2
$DAEMON_HOME/cosmovisor/upgrades/v0.3.2/bin/uptickd version --long
#commit: bde718b6b1def40da9bd79cf9872b8f1ea0b2e7d

```

## For Validators NOT Using Cosmovisor

The chain will halt at the target block. Please manually update your binary as follows:

```bash
#Download the source code and build it
git clone https://github.com/UptickNetwork/uptick.git
cd uptick
git checkout v0.3.2
make install

# View version should be v0.3.2
uptickd version --long
#commit: bde718b6b1def40da9bd79cf9872b8f1ea0b2e7d
```
